# 📅 Agenda Fácil

O **Agenda Fácil** é uma aplicação web desenvolvida para conectar clientes a prestadores de serviços de forma rápida e prática, permitindo a solicitação de orçamentos personalizados em poucos cliques.

## 🚀 Funcionalidades

- 🔍 Busca e filtro de serviços por categoria
- 🧾 Solicitação de orçamento personalizada
- 📩 Envio automático de e-mails com EmailJS
- 👨‍🔧 Direcionamento automático para prestadores por categoria
- ✅ Confirmação de envio para o cliente
- 📱 Interface responsiva com Tailwind CSS

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3 / Tailwind CSS
- JavaScript (Vanilla JS)
- EmailJS (envio de e-mails sem backend)

## ⚙️ Como funciona

1. O usuário acessa a lista de serviços
2. Seleciona ou solicita um orçamento
3. Preenche o formulário
4. O sistema envia automaticamente:
   - 📧 Um e-mail para o prestador correto (baseado na categoria)
   - 📧 Uma confirmação para o cliente (opcional)
